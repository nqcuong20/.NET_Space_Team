fonts


We have used Google Fonts in our template. You can use any google font by doing the following changes :

1. Please replace the following lines in all layout files.

              
        <http://fonts.googleapis.com/css?family=Open+Sans:400,600,300,800,700,400italic" rel='stylesheet' type='text/css'>
        <link https://fonts.googleapis.com/css?family=Roboto:400,400italic,500,500italic,700' rel='stylesheet' type='text/css'>
        <link https://fonts.googleapis.com/css?family=Poppins:400,300,500,600,700' rel='stylesheet' type='text/css'>
        <link http://fonts.googleapis.com/css?family=Montserrat:400,700' type='text/css'>

Source File (CSS)


We recommend using the FireBug add on in Firefox for a fast identification of the css properties that you might want to change.

        
          <!-- ALL CSS FILES HERE -->
          <!-- Bootstrap CSS  -->
          < rel="stylesheet" type="text/css" href="css/bootstrap.min.css">
          
          <!-- font-awesome & simple line icons CSS  -->
          < rel="stylesheet" type="text/css" href="css/font-awesome.css">
          < rel="stylesheet" type="text/css" href="css/simple-line-icons.css">
          
          <!-- owl.carousel CSS  -->
          < rel="stylesheet" type="text/css" href="css/owl.carousel.css">
          < rel="stylesheet" type="text/css" href="css/owl.transitions.css">
          
          <!-- animate CSS   -->
          < rel="stylesheet" type="text/css" href="css/animate.css">
          
          <!-- jquery-ui.min CSS   -->
          < rel="stylesheet" type="text/css" href="css/jquery-ui.css">
          
          <!-- style CSS  -->
          < rel="stylesheet" type="text/css" href="css/style.css">

javascript


This template imports the following Javascript files.

All javascript files (at js/folder) :

        
        <!-- ALL JS FILES HERE -->
        <!-- jquery js --> 
        <script type="text/javascript" src="js/jquery.min.js"></script> 
        
        <!-- bootstrap js --> 
        <script type="text/javascript" src="js/bootstrap.min.js"></script> 
        
        <!-- owl.carousel.min js --> 
        <script type="text/javascript" src="js/owl.carousel.min.js"></script> 
        
        <!-- bxslider js --> 
        <script type="text/javascript" src="js/jquery.bxslider.js"></script> 
        
        <!-- Slider Js --> 
        <script type="text/javascript" src="js/revolution-slider.js"></script> 
                
        <!-- jquery.mobile-menu js --> 
        <script type="text/javascript" src="js/mobile-menu.js"></script> 
        
        <!--jquery-ui.min js --> 
        <script type="text/javascript" src="js/jquery-ui.js"></script> 
        
        <!-- countdown js --> 
        <script type="text/javascript" src="js/countdown.js"></script> 
        
        <!--cloud-zoom js --> 
        <script type="text/javascript" src="js/cloud-zoom.js"></script> 
        
        <!-- main js --> 
        <script type="text/javascript" src="js/main.js"></script>

Overview & Features


ShopMart - Premium Bootstrap Ecommerce HTML5 Template

ShopMart is a modern, clean and professional HTML Template, It is fully responsive, it looks stunning on all types of screens and devices.

Template Features

  • HTML5 / CSS 3
  • Fully Responsive (Tested on Multiple Devices)
  • Full Width Layout
  • Cloud zoom
  • Multi-options for Home Page
  • Mega Top Menu
  • Mega Horizontal Menu
  • Mega Vertical Menu
  • Brand Slider
  • Hot Categories
  • HTML files
  • RTL support
  • List & Grid View (Products)
  • Unique & Modern Design
  • Valid HTML5 Markup
  • Owl Carousel Slider With CSS3 Transitions
  • Free Google web fonts
  • Free Icon Fonts (Font Awesome)

  • Included documentation.
